# Button #  

包括 `<button/>` 和 `<button-group/>` 标记  

## props ##  
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :---- | :---- | :---- | :---- | :---- |
| type | 类型 | string | primary / success / warning / danger / info <br/> text-primary / text-success / text-warning / text-danger / text-info | — |
| cover | text 按钮是否展示 hover 背景 | bool | true / false | false |
